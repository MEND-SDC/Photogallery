# airbnb-photogallery
This repo create the photo gallery module of airbnb.

CRUD API's

POST: /api/photos/:id
- Add an image to a property listing 

GET: /api/photos/:id 
- Get all images from a listing

PUT: /api/photos/:id 
- Update an image on a listing

DELETE: /api/photos/:id
- Remove an image from a listing
